# 🧠 MV NeuroDx - AI Brain Tumor Detection System

## 🚀 Overview
NeuroDx is an advanced AI-powered web application that analyzes brain MRI scans to detect tumors with confidence scores. It also provides smart reports and an AI assistant for medical insights.

## ✨ Features
- 🧠 MRI Scan Upload & Analysis
- 📊 Tumor Detection with Confidence %
- 📄 Report Generation
- 🤖 AI Medical Assistant (Chatbot)
- 📂 Scan History Tracking
- 🌐 Modern Dashboard UI

## 🛠️ Tech Stack
- Frontend: React.js / Next.js
- Backend: Node.js / Express
- AI Model: TensorFlow / Python
- Database: MongoDB

## 📷 Output
- Real-time MRI scan results
- Tumor detection status
- Confidence score display

## 📌 Use Case
- Hospitals & Clinics
- Medical Students
- AI Research Projects

## 📬 Future Enhancements
- Live video consultation
- Doctor appointment booking
- Multi-disease detection
